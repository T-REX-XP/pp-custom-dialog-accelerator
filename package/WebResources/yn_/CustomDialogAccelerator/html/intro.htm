Welcome to the CustomDialog Accelerator

It is a solution that aims to streamline the development of custom dialogs based on Custom page functionality by utilizing the ready-made reference dialog as a template.

How to use:

0. Create the new dialog by copying (Save As) reference dialog.

1. Make the configuration for the new dialog in the Custom Dialog Configuration table by creating a new record.

1.1 Implement the logic on the newly created dialog

2. Add a button to the ribbon using the modern ribbon editor

2.1 Populate the default parameters for the action:

  • Action: Run JavaScript
  • Library: yn_/CustomDialogAccelerator/libs/customDialogHelper.js
  • Function: yn.Navigation.showDialog
  • Parameter1 (Dialog Unique Name): yn_clonerecorddialog_984b9
  • Parameter2: SelectedControlSelectedItemReferences
  • Parameter3: SelectedControl
  • Visibility (Optional): Use a formula like this if necessary: CountRows(Self.Selected.AllItems) > 0
